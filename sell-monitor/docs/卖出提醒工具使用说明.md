# 卖出提醒与开仓检查使用说明

## 1. 配置文件位置

默认优先读取 Obsidian 里的主配置文件：

```text
E:\tools\OB\Obsidian\Trade\notes\monitor\config\sell-monitor-config.md
```

也可以通过环境变量指定别的配置文件：

```powershell
$env:SELL_MONITOR_ENV_FILE="D:\configs\sell-monitor.env"
```

## 2. Obsidian 路径配置

卖出监控输出：

```dotenv
SELL_MONITOR_OBSIDIAN_MONITOR_ENABLED=true
SELL_MONITOR_OBSIDIAN_MONITOR_DIR=E:\tools\OB\Obsidian\Trade\notes\monitor\sell
```

开仓检查输出：

```dotenv
SELL_MONITOR_OBSIDIAN_ENTRY_ENABLED=true
SELL_MONITOR_OBSIDIAN_ENTRY_DIR=E:\tools\OB\Obsidian\Trade\notes\monitor\entry
```

## 3. Watchlist 现在分成两份

卖出监控名单：

```text
E:\tools\OB\Obsidian\Trade\notes\monitor\config\sell-watchlist.md
```

开仓检查名单：

```text
E:\tools\OB\Obsidian\Trade\notes\monitor\config\entry-watchlist.md
```

对应配置项：

```dotenv
SELL_MONITOR_SELL_WATCHLIST_PATH=E:\tools\OB\Obsidian\Trade\notes\monitor\config\sell-watchlist.md
SELL_MONITOR_ENTRY_WATCHLIST_PATH=E:\tools\OB\Obsidian\Trade\notes\monitor\config\entry-watchlist.md
```

说明：

- 卖出监控只读取 `sell-watchlist.md`
- 开仓检查只读取 `entry-watchlist.md`
- 两份名单独立维护
- 同一只股票如果要同时参与两套流程，就两边都加

### watchlist 格式

```json
{
  "symbols": [
    { "symbol": "002241", "name": "歌尔股份" },
    { "symbol": "300015", "name": "爱尔眼科" },
    "300014"
  ]
}
```

说明：

- 可以只写股票代码
- 也可以同时写 `name`
- 如果配置了 `name`，输出里优先显示中文名
- 如果没配 `name`，就显示股票代码

## 4. 如何增加股票到 watchlist

直接编辑对应文件：

```text
E:\tools\OB\Obsidian\Trade\notes\monitor\config\sell-watchlist.md
E:\tools\OB\Obsidian\Trade\notes\monitor\config\entry-watchlist.md
```

## 5. 卖出监控怎么运行

单次运行：

```powershell
.\scripts\run_akshare.ps1
```

只监控一只：

```powershell
.\scripts\run_akshare.ps1 -Symbol 002241
```

每小时循环：

```powershell
.\scripts\run_akshare.ps1 -Loop -IntervalSeconds 3600
```

## 6. 开仓检查怎么运行

单次扫描整个开仓名单：

```powershell
.\scripts\scan_entry.ps1
```

只扫描一只：

```powershell
.\scripts\scan_entry.ps1 -Symbol 002241
```

每小时循环：

```powershell
.\scripts\scan_entry.ps1 -Loop -IntervalSeconds 3600
```

## 7. 历史回放

卖出回放：

```powershell
.\scripts\replay_advice.ps1 -Symbol 002241 -AsOfDate 2026-05-20
```

开仓回放：

```powershell
.\scripts\replay_entry.ps1 -Symbol 002241 -AsOfDate 2026-05-20
```

## 8. 历史回测

卖出回测：

```powershell
.\scripts\backtest_strategy.ps1 -Symbols "002241,002739" -StartDate 2025-11-20 -EndDate 2026-05-27
```

开仓回测：

```powershell
.\scripts\backtest_entry.ps1 -Symbols "002241,300015" -StartDate 2025-11-20 -EndDate 2026-05-27
```

在 PowerShell 里，多个股票代码建议加引号，避免参数拆分异常。

## 9. 输出位置

卖出监控：

```text
E:\tools\OB\Obsidian\Trade\notes\monitor\sell
```

开仓检查：

```text
E:\tools\OB\Obsidian\Trade\notes\monitor\entry
```

默认是一只股票一篇文档，最新结果放在最上面。

## 10. 交易时间限制

实时扫描默认只在 A 股常规交易时段运行：

- 09:30-11:30
- 13:00-15:00

如果只是调试，可以强制在非交易时间运行一次：

```powershell
.\scripts\run_akshare.ps1 -IgnoreTradingHours
.\scripts\scan_entry.ps1 -IgnoreTradingHours
```
