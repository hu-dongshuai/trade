# 启动开仓检查_每小时循环

对应脚本：

```text
launchers/启动开仓检查_每小时循环.bat
```

作用：

- 每 1 小时运行一次开仓检查

命令：

```bat
@echo off
setlocal
set "SELL_MONITOR_ENV_FILE=E:\tools\OB\Obsidian\Trade\notes\monitor\config\sell-monitor-config.md"
cd /d "C:\Users\admin\Documents\New project\sell-monitor"
powershell -ExecutionPolicy Bypass -File ".\scripts\scan_entry.ps1" -Loop -IntervalSeconds 3600
endlocal
```
